## Project Description

This project is a Python-based performance testing tool for the Ollama API. It measures response times and resource usage.

## Key Technologies

- Python 3
- `requests` library

## Commands

- **Run tests:** `python ollama_performance_test.py`
- **Lint:** `ruff check .`
- **Format:** `ruff format .`
